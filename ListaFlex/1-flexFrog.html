Explicação dos niveis do jogo FlexFrog

Nível CSS Aplicado Explicação
1 justify-content: flex-end; Alinha todos os itens ao final (direita) do contêiner
2 justify-content: center; Centraliza horizontalmente todos os itens
3 justify-content: space-around; Distribui espaços iguais ao redor de cada item
4 justify-content: space-between; Space uniforme apenas entre itens, sem margem nas bordas
5 align-items: flex-end; Alinha verticalmente todos os itens na parte inferior
6 justify-content: center;
align-items: center; 		Centraliza itens horizontal e verticalmente
7 justify-content: space-around;
align-items: flex-end; 		Espaça horizontalmente e alinha verticalmente ao fundo
8 flex-direction: row-reverse; Inverte a ordem dos itens na linha (direita→esquerda)
9 flex-direction: column; Altera o fluxo para coluna (itens empilhados verticalmente)
10 flex-direction: row-reverse;
justify-content: flex-end; 		Inverte linha e alinha todos os itens ao final do contêiner
11 flex-direction: column;
justify-content: flex-end; 		Empilha coluna e alinha itens ao final (embaixo)
12 flex-direction: column-reverse;
justify-content: space-between; 		Inverte coluna e distribui espaço entre itens
13 flex-direction: row-reverse;
justify-content: center;
align-items: flex-end; 		Inverte linha, centraliza e alinha vertical ao fundo
14 .yellow { order: 1; } Move o item amarelo para depois dos demais
15 .red { order: -1; } Coloca o item vermelho antes dos demais
16 .yellow { align-self: flex-end; } Alinha somente o item amarelo na parte inferior
17 .yellow { align-self: flex-end; }
#pond { order: 1; } 		Alinha o amarelo ao fundo e move o contêiner para segunda posição
18 #pond { flex-wrap: wrap; } Permite que itens quebrem linha ao não caberem
19 #pond { flex-wrap: wrap; }
flex-direction: column-reverse; 		Inverte coluna e permite múltiplas colunas com wrap
20 #pond { flex-flow: column wrap; } Combina coluna com quebra automática de linha
21 #pond { align-content: flex-start; } Alinha todo o bloco de linhas ao topo
22 #pond { align-content: flex-end; } Alinha todo o bloco de linhas à base
23 flex-direction: column-reverse;
align-content: center; 		Inverte coluna e centraliza o agrupamento de linhas
24 flex-flow: column-reverse wrap-reverse;
justify-content: center;
align-content: space-between; 		Inverte coluna e wrap, centraliza itens e distribui linhas
