Questões de 1 a 3

Questão 1:

Quando o usuário pesquisar o site, será chamando o DNS que é o nome de dominio do site (EX: www.docs.google.com) e o IP que é o endereço do usuário será enviado para o link, então com a requisição HTTP, o navegador vai procurar a requisição e formular uma resposta em HTTP, quando o redicionamento for aceito é inicializada a estilização lógica da pagina em CSS e JavaScript e o site irá abrir respondendo OK no cabeçalho HTTP, já se o redirecionamento não for aceito a pagina HTML informa qual o erro que veio na resposta HTTP (EX: Error 200)

Questão 2:

Não, o HTML não é uma linguagem de programação e sim de marcação, ele foi criado especificamente para texto na internet.

Questão 3:

A regra CSS foi criada para estilizar os textos em HTML, com ele podemos estilizar partes selecionadas do texto em vez do texto inteiro, para selecionarmos uma parte especifica do documento ultilizamos seletores com parametros correspondentes com os elementos do texto, por exemplo , para selecionar um elemento pelo seu Id se coloca '#ID' já para selecionar um elemento pela sua classe se coloca '.classe'. Após selecionar algum elemento é possivel estilizar ele de diversas formas, por exemplo, 'color : red;' deixa o texto da cor vermelha, ou 'border: 10px solid black;' que faz uma borda lisa preta em volta do elemento selecionado. Além disso também é possivel posicionar o elemento selecionado, por exemplo, aapós colocars o 'display: flex;' você pode usar comandos flex como o 'justify-content:center;' que deixa o elemento no centro horizontal.
